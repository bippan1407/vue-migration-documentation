# Installation

Installing vue-migration

::: code-group

```sh [npm]
$ npm install -g vue-migration
```

```sh [yarn]
$ yarn add -g vue-migration
```
:::

Installing vuex-to-pinia-transform

::: code-group

```sh [npm]
$ npm install -g vuex-to-pinia-transform
```

```sh [yarn]
$ yarn add -g vuex-to-pinia-transform
```
:::
